# Школа «КАЛИБР» — сайт

Next.js сайт школы стрелковой подготовки «КАЛИБР» (Воскресенск).

ТЗ: `ТЗ_КАЛИБР_v2.docx` / `ТЗ_КАЛИБР_v2.md`

## Запуск

```bash
cd calibr
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm start
```

## Функции

- Все публичные страницы по ТЗ v2 (v1 + v2 сервисы)
- Онлайн-запись со слотами → `data/bookings.json`
- Калькулятор стоимости
- Тренажёр-тест (тренировка / экзамен)
- 3D-сцены (React Three Fiber)
- Версия для слабовидящих
- 13 разделов `/docs/`
- SEO: sitemap, robots, OpenGraph

## Стек

Next.js 15 · TypeScript · Tailwind CSS · React Three Fiber · PostgreSQL-ready API
